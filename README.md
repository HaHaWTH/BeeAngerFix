## Bee Anger Fix

Fixes Bee anger never gets reset since Minecraft 1.21.11, which breaks honey farms.

## Fixed issues:

- [MC-305388](https://bugs.mojang.com/browse/MC/issues/MC-305388): bees aggression is not timed out and it is impossible to get away from them.

## Compatibility

- Fabric/Forge/NeoForge 1.21.11+
